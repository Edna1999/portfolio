Random Password Generator

This password generator was constructed using HTML, CSS and Vanilla Javascript. Prompts appear on the page with questions; your inputs will guide the generator in the type of password you'd like.

Daily Planner

This daily planner was built using HTML, CSS, vanilla Javascript and jQuery. Moment.js was used for the time and date and local storage was used so that when entries are made they are saved. It is color coded so the current hour is red, time before the current hour is grey and the time after the current hour is green.
Coding Quiz

This is a coding quiz built using HTML, CSS and Vanilla Javascript. You are given 75 seconds to answer five questions once answered a display of either correct or wrong will appear on the page, if an answer is wrong the timer will deduct fifteen seconds from your time. Local storage has been used to add up your points and at the end of the quiz once your initials are entered your score will be displayed.
